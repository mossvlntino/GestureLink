# 🖐️ GestureLink: Vision-Based Controller

## 📖 Overview
A Python-powered Human Interface Device (HID) that replaces physical mice with real-time hand gesture recognition. By implementing a two-handed modal architecture, this project solves the common UX issue of "ghost clicks" in standard gesture controllers.

## 🛠️ Focus Areas
* **Human-Computer Interaction (HCI):** Designing intuitive gestures for native OS control.
* **Computer Vision:** Utilizing MediaPipe and OpenCV for high-fidelity landmark tracking.
* **Manual QA:** Extensive physical testing to calibrate cursor smoothing and gesture reliability.
